**DASHBOARD DESIGNING**

**Description**

This project involves designing an interactive analytical dashboard using Power BI Desktop for a Smart Campus system.
The dashboard visualizes key academic metrics such as attendance percentage and academic performance, helping stakeholders monitor and analyze student data efficiently.

**Objective**

To design a Power BI dashboard for monitoring key student metrics

To visualize attendance and performance using KPI cards and charts

To enable interactive filtering by Department and Semester

**Dataset Used**

File name: 5.student\_performance.csv

Data includes:

Student\_ID

Department

Semester

Attendance\_Percentage

Assignment\_Marks

Internal Test scores

**Tools \& Technologies**

Power BI Desktop

CSV Dataset

DAX (Data Analysis Expressions)

**KPIs Implemented**

Average Attendance

Average Performance

These KPIs are calculated using DAX measures and displayed as KPI Cards.

**Visualizations Included**

KPI Cards

Average Attendance

Average Performance

Bar Chart

Attendance Percentage by Department

Line Chart

Assignment Marks by Semester

Slicers

Department

Semester

**Interactivity Features**

Slicers allow filtering data by Department and Semester

KPI cards and charts update dynamically based on slicer selection

Multiple visuals are linked for better analytical insight

**Dashboard Output**

The final dashboard displays:

Overall attendance and performance metrics

Department-wise attendance comparison

Semester-wise assignment performance trend



**Learning Outcomes**

Understanding dashboard design principles

Implementing KPIs using DAX

Creating interactive visuals using slicers

Analyzing academic data effectively

**Conclusion**

This Power BI dashboard successfully converts raw student data into meaningful insights using interactive visuals and KPIs. It supports academic monitoring, improves transparency, and aids decision-making in educational institutions.
